
# 应急响应

GB/Z 20986-2007 中，分有害程序事件、网络攻击事件、信息破坏事件、信息内容安全事件、设备设施故障、灾害性事件和其他信息安全事件7个基本类别

##  信息安全事件分类指南

GB/Z 20986 

1级 -> 特别重大事件
2级 -> 重大事件
3级 -> 较大事件

##  应急响应六个阶段

第一阶段 -> 准备
第二阶段 -> 检测
第三阶段 -> 遏制/止损
第四阶段 -> 根除
第五阶段 -> 恢复
第六阶段 -> 跟踪总结

##  计算机取证

# 灾难备份与恢复


## 基本概念
恢复点目标(RPO) -> 允许丢失的数据量

恢复时间目标(RTO) -> 从停顿到必须恢复的时间要求，容忍的信息系统和业务功能恢复的时间
